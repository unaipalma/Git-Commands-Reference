|--------------------------------------------|-------------------------------------------------------|------------------------------------------------------|

| `git clone URL`                            | Clone an existing repository.                         | `git clone https://github.com/example/project.git`   |
| `git add file_name`                        | Add changes to the staging area.                      | `git add file.txt`                                   |
